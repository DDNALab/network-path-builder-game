Path Builder: Network Design Game

Draw arrows from the source to the target through relay nodes. Source cannot connect directly to Target. Link budget depends on the number of relay nodes.

1. Set up

The blue node is the source. The red node is the target. Orange nodes are relay nodes that can pass flow along.

2. How to draw arrows

Click one node. Then click another node. An arrow will be drawn from the first node to the second node.

Rule: Source cannot connect directly to Target. Also, two nodes can have at most one link between them.

Vocabulary

node = place edge = arrow path = route source = start target = end
